# BazenBot
Automatic pool control using arduino NANO

This is my first arduino project.

Parts:

1/ Arduino NANO

2/ RTC - DS3231

3/ LCD 16x2 / I2C

4/ 2x Thermometer DS18B20

5/ 4x Buttons

6/ modul 4 relay

7/ Motorized Ball Valve,DC5V/12V 1"

8/ 3x LED

9/ 1x DUO LED

10/ 4x rezistor 330Ω

11/ 1x resitor 4k7Ω
